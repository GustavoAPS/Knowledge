HTML Basics
  • A project generally has a file called index.html, that is the ‘main’ file for the project.
  • HTML is for hypertext markup language, it is used to structure a web page.
  • Elements of a HTML page are put with tags, that look like this <name_name></tag_name>, the reason we have two of the same tag, is because the first one is the ‘opening’ one and the second with the / character is the closing one.
  • A HTML file will always have the tags <html></html> . Inside html tag we may have the language definition for the page, this help some browsers with screen readers for example to know the language of the page.
  • Then we have two other sections, the head and the body. Head is where we put the settings of the page like for example the tittle with the tittle tag. The body section is where we put elements that will be visible in the page, like texts, images or buttons.
  • In HTML elements may be nested one inside the other.
  • Elements may have attributes inside them, attributes are placed in the opening tag of the element, and give additional information about the element, the image element may have attributes defining the size of the image like the <a></a> element may have the link for the attribute being set. This looks like this: <a href="https://www.google.com">Google</a>
  • Classes and ID's are used in html to identify elements, so they can be selected in the CSS and Javascript
  • Classes can be used over many elements, but ID's can be used only once in a page, here an example of each:
    <p class="paragraph-style">
    <p id="unique-style">
HTML Boilerplate

We do not need to write all the HTML structure every time we need to write HTML code, this is why we have the boilerplate code, it is a piece of code with a stadard structure of all the basic HTML we will need by standard.

HTML Intermediate/Advanced

Some characters in HTML are reserved for the language like symbol for greater than or less than, when needed to write these characters we need to use html entities.
